Introducing Darch, a tool for building immutable, reproducable, and stateless bootable Linux images. Think Docker, but for bare-metal.

Posted on February 1, 2018

I have previously blogged about my desire to have a “docker-like” environment for build images that I can boot bare-metal. With that said, I created Darch. What it is Darch is an application (written in golang) that makes building and booting rootfs images simple. It will generate rootfs.squash files and update grub.cfg for booting them. Even though Darch supports Arch only (Gentoo/VoidLinux soon), you can build and boot the images from any operating system that has grub installed. [Read More]
